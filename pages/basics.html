Что такое JavaScript?

JavaScript — это высокоуровневый, интерпретируемый язык программирования, который выполняет код прямо в браузере пользователя. С его помощью можно:

Добавлять интерактивность

Кнопки, формы, слайдеры

Обновлять контент

Без перезагрузки страницы

Работать с данными

Отправлять и получать данные

Создавать игры

Браузерные игры и анимации

Интересный факт

JavaScript был создан всего за 10 дней в 1995 году Бренданом Айком для браузера Netscape Navigator. Несмотря на схожее название, он не имеет отношения к Java.

Первая программа

Давайте напишем нашу первую программу на JavaScript. В браузере есть консоль разработчика, куда можно выводить информацию:

Попробуйте сами:

// Это комментарий - он не выполняется
console.log("Привет, мир!");

// Можно выводить разные типы данных
console.log("Добро пожаловать в JavaScript!");
console.log(42);                 // Число
console.log(true);               // Логическое значение
console.log([1, 2, 3]);          // Массив
console.log({name: "Вася"});     // Объект

Вывод в консоль:

Нажмите "Запустить код" чтобы увидеть результат

Как открыть консоль:

  • Chrome/Firefox: F12 → вкладка Console
  • Windows: Ctrl + Shift + I
  • Mac: Cmd + Option + I

Как подключить JavaScript к HTML

Есть несколько способов подключения JavaScript к HTML-странице. Выберите подходящий для вашей задачи:

Встроенный (inline) скрипт

Подходит для небольших скриптов прямо в HTML:

<!DOCTYPE html>
<html>
<head>
    <title>Пример</title>
</head>
<body>
    <h1>Моя страница</h1>
    
    <script>
        // JavaScript код прямо здесь
        console.log("Страница загружена!");
        
        function sayHello() {
            alert("Привет!");
        }
    </script>
</body>
</html>
Не рекомендуется для больших проектов

Внешний файл (рекомендуется)

Лучшая практика - выносить JavaScript в отдельные файлы:

project/
index.html
script.js 
<!-- index.html -->
<script src="script.js" defer></script>
// script.js
console.log("Скрипт загружен!");
// Весь JavaScript код здесь

Современный подход

В современных проектах используют модули:

<!-- Тип модуль -->
<script type="module" src="main.js"></script>
// main.js
import { calculate } from './math.js';
import { formatDate } from './utils.js';

// Используем импортированные функции
console.log(calculate(10, 5));

Синтаксис JavaScript

Каждый язык программирования имеет свой синтаксис - правила написания кода.

1

Инструкции

Каждая инструкция заканчивается точкой с запятой:

let x = 5;        // Присваивание
let y = 10;       // Еще одна переменная
let z = x + y;    // Сложение
console.log(z);   // Вывод результата
2

Комментарии

Комментарии помогают понять код другим разработчикам:

// Однострочный комментарий

/* Многострочный
   комментарий
   на несколько строк */

/**
 * JSDoc комментарии
 * @param {number} a - первое число
 * @param {number} b - второе число
 * @returns {number} сумма чисел
 */
function add(a, b) {
    return a + b;
}
3

Чувствительность к регистру

JavaScript различает заглавные и строчные буквы:

let userName = "Иван";    // Правильно
let username = "Петр";    // Другая переменная!
let UserName = "Сергей";  // И третья!

console.log(userName);    // "Иван"
console.log(UserName);    // "Сергей"
// console.log(username);  // Ошибка! переменная не объявлена
Это частая причина ошибок у новичков!

Типы данных

В JavaScript данные делятся на примитивные (хранят одно значение) и ссылочные (объекты).

Number

Числа: целые и дробные

let age = 25;
let price = 99.99;
let temperature = -10;
let billion = 1e9; // 1000000000
let hex = 0xFF;    // 255 в 16-ричной

String

Текст в кавычках

let name = "Анна";
let message = 'Привет!';
let text = `Шаблонная строка`;

// Конкатенация
let greeting = "Привет, " + name + "!";
let template = `Вам ${age} лет`;

Boolean

Логические значения

let isLogged = true;
let hasAccess = false;
let isAdult = age >= 18; // true или false

// Логические операторы
let result = true && false; // false
let choice = true || false; // true
let inverse = !true;        // false

Особые значения

Undefined, null, NaN

let undefinedVar;    // undefined
let empty = null;      // намеренное отсутствие
let notANumber = NaN;  // не число
let infinity = Infinity; // бесконечность

console.log(typeof undefinedVar); // "undefined"
console.log(0 / 0);               // NaN

Практическое задание

Теперь применим полученные знания на практике:

Задача: Калькулятор возраста

Легкий уровень

Создайте программу, которая:

  1. Запрашивает имя пользователя
  2. Запрашивает год рождения
  3. Вычисляет возраст
  4. Выводит персонализированное сообщение
  5. Определяет совершеннолетний ли пользователь

Подсказки:

  • Используйте prompt() для ввода данных
  • Для вычисления возраста: текущий год - год рождения
  • Используйте console.log() для вывода
  • Условный оператор if...else для проверки возраста

Редактор кода:

Вывод вашего кода появится здесь

Пример решения:

// Решение задачи "Калькулятор возраста"

// 1. Запрашиваем данные у пользователя
let userName = prompt("Как вас зовут?");
let birthYear = prompt("В каком году вы родились?");

// 2. Вычисляем возраст
let currentYear = new Date().getFullYear();
let age = currentYear - birthYear;

// 3. Выводим приветствие
console.log("=== Результаты ===");
console.log(`Привет, ${userName}!`);
console.log(`Тебе ${age} лет.`);

// 4. Проверяем совершеннолетие
if (age >= 18) {
    console.log("Ты совершеннолетний!");
} else {
    console.log("Ты еще несовершеннолетний.");
}

// 5. Дополнительная информация
if (age >= 65) {
    console.log("Пенсионный возраст!");
} else if (age >= 60) {
    console.log("Скоро на пенсию!");
}

console.log("Спасибо за участие!");
Советы по улучшению:
  • Добавьте проверку корректности ввода
  • Используйте parseInt() для преобразования строки в число
  • Добавьте больше проверок возраста
